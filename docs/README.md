# Brick Design UI 组件库

### 快速开始

### 1.安装

```bash
npm install brick-design

OR

yarn add brick-design
```

### 2.引用组件库

```javascript
// 全部引入
import "brick-design/dist/css/index.css";
import BrickDesign from "brick-design";

Vue.use(BrickDesign);

// 按需引入
import "brick-design/dist/css/demo.css";
import { Demo } from "brick-design";

Vue.use(Demo);
```
